# Language-Specific Guidelines

このディレクトリには、AI エージェントが実装時に遵守すべき言語固有のコーディングガイドラインが格納されています。

## 仕組み

- `tdd-implementer` エージェントは実装開始時にこのディレクトリを確認します。
- `PLAN.md` の Affected Files に含まれるファイルの拡張子に対応するガイドラインが存在する場合、自動的に読み込まれます。
- Red / Green / Refactor の全フェーズでガイドラインが遵守されます。
- 対応するガイドラインが存在しない場合は、既存コードのパターンに従います。

## ガイドライン一覧

| ファイル | 言語 | 対応拡張子 |
|---|---|---|
| [php.md](./php.md) | PHP | `.php` |

## 新規言語の追加手順

1. `ai-agent/guidelines/<language>.md` を作成する。
2. 以下のセクションを含める:
   - コーディング規約（命名、スタイル等）
   - ドキュメンテーション規約（コメント、docblock 等）
   - テストコード規約
   - その他言語固有のルール
3. 本 README のガイドライン一覧に追記する。
